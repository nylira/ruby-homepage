Some Top Ruby Projects

  • Rails Rails is a web-application framework for the MVC pattern that includes both a template engine, controller framework, and object-relational mapping package.
  • Active Support Utility classes and extension to the standard library that were required by Rails, but found of general use.
  • Ruby Installer A self-contained installer that includes the Ruby language, several popular extensions, an HTML Help based API reference, and the "Book of Ruby" courtesy of Huw Collingbourne.
  • ParseTree ParseTree is a C extension (using RubyInline) that extracts the parse tree for an entire class or a specific method and returns it as a s-expression (aka sexp) using ruby's arrays, strings, symbols, and integers.
  • RubyGems RubyGems is the Ruby standard for publishing and managing third party libraries.
